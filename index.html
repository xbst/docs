List of Our Open-Source Projects

KUSBA KUSBA Pro
A USB accelerometer designed for making Klipper's input shaping much easier. It's an ADXL345 accelerometer and RP2040 MCU on the same PCB. A nozzle USB accelerometer designed for Klipper's input shaping. Unique nozzle clamp mechanism makes measuring resonances at the nozzle much easier.
Repository Store Repository Store
Nevermore PCB(s) Nevermore Sensors
PCBs designed to simplify wiring and add USB/CAN connectivity for Nevermore air filters. A tiny PCB with a BME280 (TEMP & HUM) and a SGP40 (VOC) sensor, designed for use in Nevermore air filters, Home Assistant projects and more.
Repository Store Repository Store
PiCAN ToqueCAN
A small USB drive shaped USB to CAN bus adapter PCB. A Raspberry Pi toque with, 2x CAN outputs, 3x USB ports and 5V buck converter for powering the Pi.
Repository Store Repository Store
Birds' Nest (USB) Birds' Nest CAN
Birds' Nest is a USB hub PCB designed for toolchanger printers. It features 6 USB outs for up to 6 toolheads. It also features filament sensor, RGB and thermistor ports. Birds' Nest CAN is a USB CAN hub PCB designed for toolchanger printers. It features 6 CAN ports connected to a USB CAN adapter. It also features filament sensor, RGB and thermistor ports.
Repository Store Repository Store
AFC-Lite MOAR_CAN
AFC-Lite is the Box Turtle AFC controller PCB with stepper and brushed DC motor drivers, switch and LED ports. It supports 4 lanes. MOAR_CAN is a ToqueCAN-compatible 8/9-port CAN bus hub, with better CAN topology.
Repository Store Repository Store
